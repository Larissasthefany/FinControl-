FinControl+ é uma aplicação web criada para **registrar e organizar despesas pessoais** de forma simples e visual.

O objetivo do projeto é ajudar usuários a **terem mais controle sobre suas finanças**, permitindo registrar gastos, visualizar informações e acompanhar pagamentos.

Atualmente o projeto possui **estrutura HTML e estilização em CSS**, enquanto a lógica em JavaScript será implementada nas próximas etapas.

---


# 🚀 Tecnologias utilizadas

* **HTML5** → estrutura da aplicação
* **CSS3** → estilização e layout
* **Google Fonts** → tipografia (Poppins e Inter)

JavaScript já está preparado na estrutura do projeto e será utilizado para implementar as funcionalidades.

# 📝 Formulário de despesas

O formulário permite registrar informações como:

* **Description** → descrição da despesa
* **Total Value** → valor total da compra
* **Date of Purchase** → data da compra
* **Due Date** → data de vencimento
* **Amount of Installments** → quantidade de parcelas

⚠️ No momento os dados **ainda não são processados**, pois a lógica em JavaScript será implementada posteriormente.
